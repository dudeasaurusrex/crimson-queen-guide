# Crimson Queen Necromancer Guide

A mobile-friendly, offline-capable version of the [Crimson Queen (Necromancer)](https://steamcommunity.com/sharedfiles/filedetails/?id=1179657481) build guide for Divinity: Original Sin 2.

## Features
- **Mobile-First Design:** Optimized for small screens and touch.
- **Offline Capable:** Single HTML file that works anywhere.
- **Progress Tracking:** Includes a leveling checklist that saves to your browser's local storage.
- **Dark Mode:** Easy on the eyes for late-night gaming sessions.

## How to use
Simply open `index.html` in any web browser. 

If hosted on GitHub Pages, it will be available at your repository's URL.
